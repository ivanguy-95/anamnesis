# Анамнез спортсмена

Адаптивный веб-опросник для сбора спортивно-медицинского анамнеза. На выходе — JSON-профиль и PDF-документ, который подаётся как вход для генерации персональной фарм-схемы.

## Стек

Vanilla HTML / CSS / JavaScript (ES-модули). Без сборщика, без зависимостей. Хранение прогресса — `localStorage`. Адаптивные подсказки — Claude API (подключим позже).

## Структура

```
Анамнез/
├── index.html      # точка входа: HTML-shell, подгрузка модулей
├── styles.css      # дизайн-токены и базовые стили
├── app.js          # инициализация, навигация между блоками
├── state.js        # JSON-профиль + localStorage (load/save/get/set/export)
├── blocks/         # модули блоков опросника (по одному файлу на блок)
└── README.md
```

## Запуск локально

ES-модули требуют HTTP-сервер (не работают по `file://`). Самый простой вариант:

```bash
cd Анамнез
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

## Статус

Каркас приложения собран — пустой shell, схема JSON-профиля и пустая папка `blocks/` для будущих модулей. Блоки добавляются итерационно по конкретным запросам.
