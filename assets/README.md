# Папка для логотипа и брендинга

Положи сюда файл логотипа — он автоматически появится в **левом верхнем углу** сайта
(и на десктопе, и на телефоне).

## Куда и как назвать

Перетащи файл логотипа прямо в эту папку и назови его **`logo`** с нужным расширением:

```
Анамнез/assets/
└── logo.svg     ← лучший вариант
```

Сайт сам подхватывает первый найденный файл в таком порядке:
`logo.svg` → `logo.png` → `logo.jpg` → `logo.webp`.
То есть можно положить любой из них — например `logo.png`.

## Какой формат и размер

- **`.svg`** — идеально: масштабируется без потери чёткости на любых экранах.
- **`.png`** (с прозрачным фоном) — тоже отлично. Высота оригинала от ~80–120 px.
- `.jpg` / `.webp` — сработают, но без прозрачности.

> На сайте логотип отображается высотой ~36 px (десктоп) / ~30 px (телефон),
> ширина — автоматически по пропорциям. Лучше горизонтальный или квадратный лого.

## Что дальше

Когда файл будет в папке — просто обнови страницу (`Cmd+Shift+R`). Если хочешь
поменять размер или отступы логотипа — скажи, подправлю.
