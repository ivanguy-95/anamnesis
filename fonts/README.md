# Папка для шрифтов

Положи сюда файлы шрифта — и скажи мне «шрифты на месте», я подключу их в `styles.css`.

## Куда класть

Просто перетащи файлы шрифта прямо в эту папку:

```
Анамнез/fonts/
├── МойШрифт-Regular.woff2
├── МойШрифт-Medium.woff2
├── МойШрифт-SemiBold.woff2
└── МойШрифт-Bold.woff2
```

## Какие форматы подойдут

Работает любой из этих (в порядке предпочтения):

- **`.woff2`** — лучший вариант: самый лёгкий, грузится быстрее всего. Если есть выбор — давай его.
- **`.woff`** — тоже отлично.
- **`.ttf` / `.otf`** — сработают, но весят больше. Если только они — ничего страшного, подключу как есть.

> Можно положить несколько форматов одного шрифта — браузер сам выберет лучший.

## Сколько начертаний нужно

В макете используются такие толщины:

| Где | Толщина (font-weight) |
|-----|----------------------|
| Обычный текст, подписи полей | 400 (Regular) |
| Метки, радио-кнопки | 500 (Medium) |
| Кнопки, заголовки секций | 600 (SemiBold) |
| Крупные заголовки экранов | 700 (Bold) |

**Идеально** — прислать 4 файла (Regular / Medium / SemiBold / Bold).
**Минимум** — хватит и одного Regular: остальные толщины браузер «дорисует» сам (выглядит чуть хуже, но работает).

## Что дальше

Когда файлы будут в папке, напиши мне название шрифта (или просто «готово») —
я:
1. пропишу `@font-face` для каждого файла,
2. переключу `--font-sans` в `styles.css` на новый шрифт,
3. закоммичу и задеплою — проверишь на живом сайте.
